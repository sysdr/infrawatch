# Day 60: Advanced Analytics System

Comprehensive analytics with statistics, anomaly detection, predictions, and correlations.

## Quick Start

### Without Docker
```bash
./build.sh
```

### With Docker
```bash
docker-compose up --build
```

## Access
- Dashboard: http://localhost:3000
- API: http://localhost:8000
- API Docs: http://localhost:8000/docs

## Features
- Real-time statistical analysis
- Multi-method anomaly detection
- Predictive forecasting
- Correlation analysis

## Testing
```bash
cd tests && python3 -m pytest test_analytics.py -v
```
